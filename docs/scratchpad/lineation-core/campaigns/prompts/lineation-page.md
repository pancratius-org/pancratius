You decide, per keyed body line of a Russian spiritual book, PROSE vs LINEATED. The author ended
almost every line with Enter, so a line being on its own row reveals nothing — prose and lineated
look identical in the markup. He shows intent by how he STYLES paragraphs on the page: prose set
DENSE (tight, lines that fill and run on), lineated set APART (spacing, indentation, short lines that
stop early). He does this DIFFERENTLY per document — judge THIS page on its own terms.

  prose     — the break is just the Enter habit; joining the lines into a paragraph loses nothing,
              the text reads as continuous, developing sentences.
  lineated  — the break is INTENDED; joining would DAMAGE the reading — its cadence, pairing,
              emphasis, list-shape.
(There is no "verse" label — decide only prose vs lineated.)

UNIT BEFORE LINE. First see the smallest DISPLAY UNIT on the page — prose paragraph, numbered/titled
item, list, litany, stanza-like run, dialogue turn — and label lines by what that unit is doing,
unless the page clearly changes register inside it. For a numbered/titled item ("1. Title" with its
following text shown as ONE displayed block), the title/body break is intentional lineation even
though the body wraps.

PROSE GUARDRAIL — do NOT over-lineate. A unit whose body is a single FLOWING sentence that WRAPS at
the reading column is PROSE, even when it carries a speaker prefix or label (e.g. "Панкратиус: мы
привыкли смотреть на вещи…") — the prefix does not make it lineated. Whole complete sentences laid
one per line, even with anaphora (repeated "Ты…"), are a paragraph, not a litany. Choose prose
whenever the break does no visible reader-facing work; "it would read fine as a paragraph" supports
prose.

TOWARD LINEATED (strongest first): a run of SHORT nowrap lines that stop early by choice — none
reaching the column where text would wrap; this holds EVEN for plain narrative or dialogue. A single
sentence deliberately split across lines. A SHORT standalone title / lead-in / speaker turn that
stops early ("Светозар сказал:"). Repeated short members after a colon or dash. A contrast split
across lines ("Не с неба — / а из сердца"). Broken vows / prayers / invocations; runs set off by
stanza gaps. "It reads MORE TRUE broken than joined" supports lineated.

EVIDENCE. The authored DOCX PAGE image is the AUTHORITY for visual intent — read the whole picture:
does this block sit DENSE as a paragraph, or APART as standalone lines? (Raw row position alone is
not lineation — every line is its own paragraph by the Enter habit.) The LISTING gives each line's
key, exact text, and a WRAPS / nowrap flag; use it to identify which line you label and for content
cues, but when it conflicts with the page the PAGE decides the verdict (it is not lossy; the listing's
grouping can be mis-extracted). Hard markers (heading / *** / image / blank / right-aligned /
blockquote) are separators — a run never crosses one and you never label them. Calibrate to THIS
page; do not import a rule from another book.

exact key.
