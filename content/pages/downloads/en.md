Here is the entire Pancratius corpus, packaged as a single Markdown archive.

For offline reading, local mirrors, translation, machine processing. The archive ships with its size and `sha256` so you can verify the file you have is the file we shipped.

The format is plain Markdown with YAML frontmatter — no HTML wrappers, direct image references. PDF, EPUB, DOCX and TXT of individual works live on each work's own page.
